End-to-End Supply Chain Analytics & Delivery Delay Prediction System

Project Overview :
An end-to-end supply chain analytics project focused on data cleaning, SQL analysis, exploratory data analysis, demand forecasting, and delivery delay prediction.

Objectives :
Analyze supply chain operations
Identify delivery delay drivers
Generate business insights
Forecast future demand
Support data-driven decision making

Tech Stack : Python, Pandas, NumPy, Scikit-Learn, SQL, Power BI

Project Structure :
Data Cleaning.py
EDA.py
SQL Analysis.sql
Prediction Model.py
Forecast_Output.csv
prediction_results.csv
feature_importance.csv

Key Features :
Data Cleaning & Preprocessing
Exploratory Data Analysis
SQL Business Analysis
Delivery Delay Prediction
Demand Forecasting
Feature Importance Analysis

Business Impact :
The project helps reduce logistics risks, improve delivery performance, optimize inventory planning, and enhance operational efficiency.
